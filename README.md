## Project Overview

This project analyzes employee attrition patterns using HR data and provides insights to help organizations improve employee retention. An interactive Power BI dashboard was developed to visualize key attrition trends and support data-driven decision-making.

## Objectives

* Analyze employee attrition across departments, age groups, education levels, and salary bands.
* Identify major factors contributing to employee turnover.
* Develop a dashboard for HR decision-making.
* Support analysis with a predictive Logistic Regression model.

## Tools Used

* Power BI
* Python
* Pandas
* Scikit-Learn
* Microsoft Excel

## Dashboard KPIs

- Employee Count: 1470
- Attrition Count: 238
- Attrition Rate: 16.08%
- Average Age: 36.92
- Average Salary: 6.50K
- Average Tenure: 7.01 Years

## Key Findings

* Overall attrition rate: 16.08%
* Employees aged 26–35 showed the highest attrition.
* Lower salary bands experienced higher employee turnover.
* Sales and Research & Development departments contributed the most attrition.
* Frequent business travel was associated with increased attrition.
* Employees with shorter tenure were more likely to leave the organization.

## Machine Learning Results

A Logistic Regression model was developed to predict employee attrition.

* Accuracy: 84.46%
* Confusion Matrix generated
* Classification Report generated

## Recommendations

* Improve employee engagement initiatives.
* Review compensation structures for lower salary groups.
* Provide better career growth and promotion opportunities.
* Monitor employee workload and business travel frequency.
* Strengthen retention strategies for new employees.

## Conclusion

The project successfully identified the primary factors influencing employee attrition and provided actionable recommendations to improve workforce retention and organizational performance.
